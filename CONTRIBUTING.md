# Contributing to Wangbar

Thank you for your interest in contributing to Wangbar!

## Pull Request Process

This repository requires pull requests to be approved before merging to the main branch.

### How to Contribute

1. **Fork the repository** or create a new branch
   ```bash
   git checkout -b feature/your-feature-name
   ```

2. **Make your changes** in your branch
   - Follow the existing code style
   - Test your changes thoroughly

3. **Commit your changes**
   ```bash
   git add .
   git commit -m "Description of your changes"
   ```

4. **Push to your branch**
   ```bash
   git push origin feature/your-feature-name
   ```

5. **Create a Pull Request**
   - Go to the repository on GitHub
   - Click "New Pull Request"
   - Select your branch
   - Provide a clear description of your changes
   - Submit the pull request

6. **Wait for Review**
   - @AwangWOW will be automatically added as a reviewer
   - Address any feedback or requested changes
   - Once approved, your changes will be merged

### Branch Protection

- Direct pushes to the `main` branch are not allowed
- All changes must go through pull requests
- Pull requests require approval before merging
- See `.github/BRANCH_PROTECTION.md` for setup details

### Code Owners

All changes are automatically reviewed by @AwangWOW (defined in `.github/CODEOWNERS`)

## Questions?

If you have questions about contributing, feel free to open an issue!
